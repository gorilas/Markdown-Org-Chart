\# Estilos

Los fundamentos sobre los que se construye el sistema de diseño. Utiliza estas bases para conseguir una apariencia común y coherente.

\## \[Color\](estilos-color.html)

Paletas de colores accesibles para elementos neutros, de interacción, soporte o para gráficas.

\## \[Espaciado\](estilos-espaciado.html)

Valores predefinidos que se han de usar para saltos entre objetos, padding, márgenes y dimensiones.

\## \[Retícula\](estilos-reticula.html)

Combinaciones basadas en una retícula de 4 columnas para escritorio y 2 para móviles.

\## \[Tipografía\](estilos-tipografia.html)

Los tamaños y formatos de letra definidos sirven para jerarquizar el contenido y organizar la información.

\## \[Lenguaje claro\](estilos-lenguaje.html)

Intentamos comunicarnos de manera cercana y comprensible siguiendo una serie de pautas de estilo.

\## \[Uso de imágenes\](estilos-imagenes.html)

El uso de imágenes y objetos multimedia está limitado por una serie de guías de estilo.
